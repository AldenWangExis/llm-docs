## 关于本教程

面向有技术背景、无AI/DL知识的开发者。用工程思维理解LLM，专注实践应用。

**前置要求**：熟悉编程（Python优先）、理解系统架构、能访问技术资源

**学习路径**：概念 → 模型 → 工具 → 进阶 → 实战
