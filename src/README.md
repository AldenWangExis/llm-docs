## 关于本教程

AI不是许愿机，只是在海量数据训练后学会"猜下一个词"的猴子打字机。对开发者来说，问题从"如何训练AI"变成了"如何使用AI能力完成任务"，从底层算法退回到了熟悉的工程领域：数据流转、异常处理、成本优化。

**前置要求**：编程能力、系统架构思维、技术资源访问能力

**学习路径**：概念认知 → LLM工程 → 工具链 | 进阶按需
