# 第二部分：LLM工程 - 从玩具到生产

掌握了 LLM、RAG、Agent 的基本概念后，5分钟就能搭建出一个可运行的原型。但从原型到生产环境，需要面对模型选型、性能优化、数据管理、系统监控等一系列工程问题。本章深入这些工程细节，讨论如何构建稳定、可维护的 LLM 应用。

::: tip 工程实践导向
本部分聚焦架构设计与工程权衡，不涉及模型训练和底层算法实现。
:::
