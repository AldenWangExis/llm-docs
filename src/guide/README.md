# LLM工程实践教程

从确定性编程到概率性工程的跨越。

## 核心理念

将概率性的AI组件封装在确定性的工程架构中。用分布式系统、API调用、架构模式理解LLM，聚焦"道"与"法"，而非琐碎的实现细节。
