# LLM工程实践教程

面向有技术背景、无AI/DL知识的开发者。

## 教程定位

- **目标受众**：有编程能力和系统架构思维，但没有AI/ML/DL理论基础
- **核心理念**：用计算机工程类比理解LLM，工程实践导向
- **学习路径**：概念重构 → 模型生态 → 工具链 → 进阶能力 → 实战案例

## 学习建议

1. **必读**：第一、二、三部分（概念、模型、工具）
2. **按需**：第四部分根据深入程度选择性阅读
3. **实践**：第五部分选择与业务相关的案例动手实现

## 前置要求

- 熟悉至少一门编程语言（Python优先）
- 理解基本的系统架构概念
- 能够访问Google/GitHub/X等技术资源
